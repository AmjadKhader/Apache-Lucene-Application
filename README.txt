This is a sample Lucene application for:
1- Create new documents
2- Create new index
3- Query indexes using the 7 queries types
4- Update documents
5- Delete documents

This application is built using apache-lucene-analyzer,  lucene-analyzer-common-8.11.1, lucene-core-8.11.1, and lucene-queryparser-8.11.1.

Design patterns used:
1- Singleton pattern
2- Factory pattern
3- Chain of responsibilities
